# Sweet Delights Bakery

This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Create a `.env.local` file and configure your environment variables.
3. Run the app:
   `npm run dev`
